# Banking System

Simple banking system made with Python.

This project allows you to:

- Create accounts
- Deposit money
- Withdraw money
- Transfer money between accounts
- Delete accounts
- Search accounts
- View statistics
- Save data using JSON

## Why I Built This

I built this project to practice:

- Functions
- Dictionaries
- Nested dictionaries
- JSON files
- Input validation
- Program logic

## Example

```text
1. Create Account
2. View Accounts
3. Search Account
4. Deposit
5. Withdraw
6. Transfer Money
7. Delete Account
8. Statistics
9. Logout
```

## What I Learned

The hardest part for me was working with nested dictionaries and calculating statistics from stored data.

I also learned how to save and load data using JSON so the information remains available after closing the program.
